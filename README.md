# HappyRobot FDE Challenge - Kanban Board

Interactive Kanban board for tracking the HappyRobot FDE Challenge progress.

## Features
- Drag & drop cards between columns
- Click cards to expand/collapse details
- Search across all cards
- Collapse/expand columns
- 48 items across 5 columns: Done, In Progress, To Do, V2/Phase 2, Nice to Have

## Deploy to Railway

1. Push this folder to a GitHub repo (or add to your existing repo)
2. Go to [railway.app](https://railway.app) and sign in with GitHub
3. Click **New Project** → **Deploy from GitHub repo**
4. Select this repo
5. Railway auto-detects the Dockerfile and deploys
6. Click **Settings** → **Networking** → **Generate Domain** to get your public URL

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000
